## 快速启动

- 安装 pnpm

```
npm install pnpm -g
```

- 安装依赖

```
pnpm install
```

- 打包组件库与工具库

```
pnpm run build:easyest
```

- 启动测试项目

```
pnpm run easyest:dev
```

- 启动静态文档站点

```
pnpm run docs:dev
```
